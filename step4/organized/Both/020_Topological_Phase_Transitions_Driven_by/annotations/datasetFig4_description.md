
# Dataset for Figure 4

## Description
This dataset contains the numerical data required to generate Figure 4 of the manuscript,  "*Slow and fast topological dynamical phase transitions in a Duffing resonator driven by two detuned tones*".
The data describes the phase-space dynamics of a driven, damped Duffing oscillator under a two-tone driving scheme.

The dataset is organized into three main components:
1.  **Phase-Space Trajectories:** Time-evolution of the system's quadratures (`u`, `v`).
2.  **Background Vector Fields:** The underlying flow fields for stream plots at specific phase offsets.
3.  **Attractor Locations:** The coordinates of stable (sinks) and unstable (saddles) steady-state solutions.

## Data Set Structure

All quadrature data (`u`, `v`, `U`, `V`, `dU`, `dV`) are given in units of **millivolts (mV)**.

### 1. Phase-Space Trajectories

These files contain the full trajectories of the system's quadratures (`u`, `v`) as they evolve in time.

-   **File:** `trajectory_ad.txt`
    -   **Description:** The trajectory corresponding to a slow, adiabatic-like sweep of the secondary drive's phase.
    -   **Format:** A 2xN matrix.
        -   Row 1: `u` quadrature (x-coordinate)
        -   Row 2: `v` quadrature (y-coordinate)

-   **File:** `trajectory_fa.txt`
    -   **Description:** The trajectory corresponding to a faster sweep of the secondary drive's phase.
    -   **Format:** A 2xN matrix.
        -   Row 1: `u` quadrature (x-coordinate)
        -   Row 2: `v` quadrature (y-coordinate)

### 2. Background Vector Fields (for Stream Plots)

This data represents the vector field (`dU`, `dV`) on a 2D grid (`U`, `V`) for four distinct phase offsets (`蠒`) between the two driving tones. These values correspond to the four panels in Figure 4, ordered from left to right.

For each phase value, four `.txt` files are provided. The phase value is indicated in the filename (e.g., `phi_0_55pi.txt`).

-   **Files for `蠒 = 0.55蟺` (Panel 1):**
    -   `flow_U_phi_0_55pi.txt`: The `U` (x-axis) grid coordinates.
    -   `flow_V_phi_0_55pi.txt`: The `V` (y-axis) grid coordinates.
    -   `flow_dU_phi_0_55pi.txt`: The `dU` (x-component) of the vector field.
    -   `flow_dV_phi_0_55pi.txt`: The `dV` (y-component) of the vector field.
    -   **Usage:** In plotting software (e.g., Python's Matplotlib), this data can be used as `streamplot(U, V, dU, dV)`.

-   **Files for `蠒 = 0.85蟺` (Panel 2):**
    -   `flow_U_phi_0_85pi.txt`, `flow_V_phi_0_85pi.txt`, etc. (Same structure)

-   **Files for `蠒 = 1.70蟺` (Panel 3):**
    -   `flow_U_phi_1_70pi.txt`, `flow_V_phi_1_70pi.txt`, etc. (Same structure)

-   **Files for `蠒 = 1.95蟺` (Panel 4):**
    -   `flow_U_phi_1_95pi.txt`, `flow_V_phi_1_95pi.txt`, etc. (Same structure)

### 3. Attractor Locations

These files contain the phase-space coordinates of the stable and unstable fixed points (attractors) for each of the four phase offsets.

-   **Files for `蠒 = 0.55蟺` (Panel 1):**
    -   `attractors_stable_phi_0_55pi.txt`: Coordinates of the stable attractors (sinks).
    -   `attractors_unstable_phi_0_55pi.txt`: Coordinates of the unstable attractors (saddles).
    -   **Format:** A 2xN matrix, where N is the number of attractors.
        -   Row 1: `u` quadrature (x-coordinate)
        -   Row 2: `v` quadrature (y-coordinate)

-   **Files for other phase values (`0.85蟺`, `1.70蟺`, `1.95蟺`):**
    -   The file structure is identical, with the phase value updated in the filename.

## Methodology

The data was generated by numerically solving the harmonic equations of motion for a driven, damped Duffing oscillator using the **HarmonicBalance.jl** package in Julia (https://quantumengineeredsystems.github.io/HarmonicBalance.jl/stable/).


The trajectories, vector fields, and attractors were computed using the normalized values of the following experimental parameters within the HarmonicBalance framework:

-   Natural Frequency (`蠅0`): `6.96986322513185e+06/(2*pi)` Hz
-   Nonlinearity (`尾`): `-1.89e17` V^-2 s^-2
-   Primary Drive Amplitude (`F鈧乣): `140`mV
-   Secondary Drive Amplitude (`F鈧俙): `30`
-   Damping (`螕`): `~110` Hz
-   Drive Detuning (`螖10`): `~-148` Hz
-   Secondary Detuning (`螖21`): `10` Hz (slow) and `90` Hz (fast)

The background vector fields and attractor locations were determined by solving the steady-state harmonic equations for the `蠒` values specified in the filenames. The trajectories were found by time-evolving the harmonic equations with a linear sweep of `蠒`.
