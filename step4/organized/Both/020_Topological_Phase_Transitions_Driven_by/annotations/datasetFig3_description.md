# Dataset for Figure 3: Topological Phase Diagram

## Description
This dataset contains the numerical data required to generate the topological phase diagram in Figure 3 of the manuscript, "*Slow and fast topological dynamical phase transitions in a Duffing resonator driven by two detuned tones*".

The data represents the calculated "net chirality" (`χ`) of the two-tone driven Duffing resonator. The net chirality is mapped as a function of the relative amplitude `h = F₂/F₁` and the phase `φ` between the two drives. The value of the chirality corresponds to the system's stability:
-   `χ = 1`: Corresponds to a monostable phase with only the high-amplitude solution.
-   `χ = 0`: Corresponds to a bistable phase where both high- and low-amplitude solutions are present.
-   `χ = -1`: Corresponds to a monostable phase with only the low-amplitude solution.

The data is provided on a 2D Cartesian grid suitable for generating a contour plot.

## Data Set Structure

The dataset consists of three `.txt` files, each containing a 501x501 matrix. The `x` and `y` coordinates are derived from the physical parameters `h` and `φ` via a polar-to-Cartesian transformation (`x = h*cos(φ)`, `y = h*sin(φ)`).

-   **File:** `phasediagram_x_fig3_hcos.txt`
    -   **Description:** The x-axis grid coordinates for the contour plot.
    -   **Format:** A 501x501 matrix representing `h * cos(φ)`.
    -   **Units:** Dimensionless.

-   **File:** `phasediagram_y_fig3_hsin.txt`
    -   **Description:** The y-axis grid coordinates for the contour plot.
    -   **Format:** A 501x501 matrix representing `h * sin(φ)`.
    -   **Units:** Dimensionless.

-   **File:** `phasediagram_netchirality.txt`
    -   **Description:** The calculated net chirality `χ` at each point on the grid.
    -   **Format:** A 501x501 matrix containing integer values of 1, 0, or -1.
    -   **Units:** Dimensionless.

-   **Usage:** In plotting software (e.g., Python's Matplotlib), this data can be used to generate the figure with a command like `contourf(x, y, netchirality)`.

## Methodology

The data was generated by numerically solving the steady-state harmonic equations of motion for a driven, damped Duffing oscillator using the **HarmonicBalance.jl** package in Julia (https://quantumengineeredsystems.github.io/HarmonicBalance.jl/stable/).

The net chirality was calculated by sweeping the parameters `h` and `φ` while keeping the primary drive and system parameters fixed. The key experimental parameters used for this calculation are:

-   Primary Drive Amplitude (`F₁`): `140 mV`
-   Drive Detuning from Resonance (`Δ₁₀/2π`): `≈ -148 Hz`
-   Resonator Natural Frequency (`f₀`): `≈ 1.11 MHz`
-   Resonator Damping (`γ/2π`): `≈ 110 Hz`
